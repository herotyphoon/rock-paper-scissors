# Rock, Paper & Scissors

This is a simple javascript project that utilizes the understanding of *conditionals*, *functions*, *objects*, *DOM* and *localStorage*.

Some features of it are:
- It shows the move to play and moves played in form of images.  
- It also keeps a counter of wins, losses and ties.  
- You can also reset the score.  
- This also stores your score counter so that you can track it anytime.
